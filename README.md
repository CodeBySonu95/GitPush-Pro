GitPush Pro ek powerful aur user-friendly mobile tool hai jo developers ko bina PC ke seedha Android device se GitHub repositories manage karne aur files commit karne ki permission deta hai. Agar aap ek mobile-first developer hain, toh ye app aapke workflow ko super fast bana degi!
✨ Features
📟 Mobile Terminal & Activity: Real-time terminal output ke saath apne uploads aur sync process ko track karein. Bilkul PC jaisa experience!
📂 Easy File Management: APKs, .swb (Sketchware) files, aur images ko directly select karke staging area mein add karein.
🔐 Secure GitHub Connection: Personal Access Tokens (PAT) ka use karke apni repositories ko safely connect karein. No password sharing needed.
🔄 Repository Switching: Ek hi click mein alag-alag projects aur repositories ke beech switch karein.
🎨 Material 3 UI: Ek modern, clean aur dark-themed interface jo use karne mein fast aur efficient hai.

🚀 Setup & Usage
Follow these simple steps to get started:
Generate Token: Apne GitHub account ki settings mein jaakar ek Personal Access Token (repo scope) generate karein.
Connect Repo: GitPush Pro app open karein, apna Token aur Repository URL enter karein aur Connect par click karein.
Add Files: Jo files ya folders aapko upload karne hain, unhe file picker se select karein.
Commit & Push: Ek meaningful commit message likhein aur Push button daba dein. Aapka code direct GitHub par live ho jayega!
