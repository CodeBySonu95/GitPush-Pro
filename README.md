## ūüďĆ What is GitPush Pro?

**GitPush Pro** ek powerful Android app hai jo aapko seedha apne phone se GitHub repositories manage karne ki taaqat deta hai ‚ÄĒ bina kisi PC, laptop, ya terminal ke.

Agar aap **Sketchware Pro**, **ACode**, ya kisi bhi mobile IDE se develop karte hain ‚ÄĒ toh ye app aapka **missing link** hai jo aapke phone ko ek complete development workstation bana deta hai.

## ūüöÄ Setup & Usage

### Step 1 ‚ÄĒ GitHub Token Generate Karo
> `GitHub Settings` ‚Üí `Developer Settings` ‚Üí `Personal Access Tokens` ‚Üí `repo` scope enable karo

### Step 2 ‚ÄĒ App Mein Connect Karo
> App kholo ‚Üí Token + Repository URL enter karo ‚Üí **Connect** tap karo

### Step 3 ‚ÄĒ Files Add Karo
> File picker se apni APK / `.swb` / koi bhi file select karo

### Step 4 ‚ÄĒ Commit & Push!
> Meaningful commit message likho ‚Üí **Push** karo ‚Üí ūüéČ Code live on GitHub!
